## 🏗️ Implementasi OOP (Object-Oriented Programming)
Proyek ini dibangun dengan menerapkan prinsip dasar pemrograman berorientasi objek untuk menjaga kode tetap modular dan *scalable*:

*   **Class & Object:** Menjelaskan blueprint utama, misalnya class `User`, `Admin`, atau `Transaction`.
*   **Encapsulation:** Penggunaan *private/protected fields* untuk melindungi data sensitif di dalam class.
*   **Inheritance:** Pewarisan sifat dari class induk ke class anak (misalnya class `Pasien` mewarisi properti dari class `Orang`).
*   **Polymorphism:** Penggunaan metode yang sama untuk perilaku yang berbeda pada objek yang berbeda.
*   **Abstraction:** Menyederhanakan kompleksitas dengan hanya menampilkan fungsionalitas yang diperlukan melalui *interface* atau *abstract class*.
